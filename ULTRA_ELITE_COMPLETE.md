## 🚀 Quick Start

```bash
# 1. Clone and setup
cd fam-tree-bot
cp .env.example .env
# Edit .env with your BOT_TOKEN

# 2. Run with Docker
docker-compose up -d

# 3. Or run manually
pip install -r requirements.txt
python src/main.py
```
